# Данная документация прилагается к мобильному приложению ТеремOk

## Поле "config":
Данное поле содержит в себе обобщенные параметры для устройства, а также его категорию.

<Error>
Важно! В конфиге должны присутствовать все указаные ниже параметры!
</Error>

```json
"config": {
    "hasChart": false,
    "hasLimits": false,
    "category": "state"
}
```
```**category** - параметр с помощью которого происходит идентификация устройства внутри приложения```
```**hasChart** - параметр отвечающий за наличие графика на странице устройства```
```**hasLimits** - параметр отвечающий за наличие у устройств пределов главного значения```

<Warning>
Параметр "category" будет описан в каждом устройстве отдельно 
</Warning>
